# Kairos Product Book

Welcome to the Kairos Product Book. This directory serves as the high-level product blueprint for the **Kairos Decision Operating System (DOS)**. 

While the `docs/architecture/` directory defines *how* the system is built, and `docs/domain/` defines *what* the system's core entities are, this Product Book defines **why** the system exists, where it is going, and how we measure success.

## Contents

1. [North Star Vision](./north-star.md): The overarching mission statement and guiding philosophy.
2. [Supported Platforms](./platforms.md): Target environments and deployment profiles.
3. [Product Roadmap](./roadmap.md): Strategic milestones and upcoming features.
4. [Glossary](./glossary.md): Standardized terminology across the product and engineering teams.
5. [Release Process](./release-process.md): How we safely deliver updates to Operators.

## Audience

This book is intended for Product Managers, Engineering Leads, and Operators seeking to understand the strategic direction of Kairos DOS.
